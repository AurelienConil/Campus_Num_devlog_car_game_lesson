# Campus Numérique - Dev Log - Car Game  

Bienvenue dans le projet **Car Game Lesson** !

Ce projet est pensé pour faire découvrir les premieres lignes de codes. Ce n'est pas un sujet au format live coding. 

## Objectif pédagogique

L'objectif est d'apprendre progressivement les bases du code à travers la création d'un petit jeu de voiture. Variables, fonctions, if.
Il n'y a pas de tableau, pas d'argument de fonction, pas de retours

## Organisation du projet

Le projet est structuré en plusieurs branches Git, chacune correspondant à une étape d'apprentissage :

- **etape1** : Mise en place du fond animé
- **etape2** : Ajout du personnage
- **etape3** : Ajouts des voitures
- etc.

Pour progresser, change simplement de branche (`git checkout etapeX`) et découvre le code associé à chaque étape.

## Fichier principal

Le fichier principal du projet est :

- `car_game_lesson.pde`

A ouvrir avec le logiciel Processing ( v3 ). Il support très bien le remplacement à chaud des fichiers.
